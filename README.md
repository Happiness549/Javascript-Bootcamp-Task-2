# Student Grading System

## 📌 Project Description
This is a simple web application built using HTML, CSS, and JavaScript.  
It allows users to enter a student’s name and mark, then calculates and displays whether the student has passed or failed instantly without reloading the page.

---

## 🚀 Live Demo
You can view the live project here:  
https://students-mark-bootcamp-2.netlify.app/

---

## ✨ Features
- Add student name and mark
- Validate input (0–100 range and empty fields)
- Calculate grade using conditions
- Display result instantly on the page
- Keep a running list of all submitted students

---

## 📊 Grading Scale
- 80 – 100 → Distinction  
- 65 – 79 → Merit  
- 50 – 64 → Pass  
- 0 – 49 → Fail  

---

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript (DOM Manipulation)

---

## ⚙️ How It Works
1. User enters student name and mark
2. JavaScript validates the input
3. Grade is calculated using if / else conditions
4. Result is displayed on the page
5. Student is added to the running list
